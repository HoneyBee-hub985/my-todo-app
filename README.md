# My Todo List

React를 사용하여 구현한 할 일 관리 애플리케이션입니다.

## 구현 기능
- ✅ 할 일 추가 (빈 내용 입력 방지)
- ✅ 할 일 목록 표시
- ✅ 완료 상태 토글 (체크박스 + 시각적 표시)
- ✅ 할 일 삭제
- ✅ 반응형 디자인

## 컴포넌트 구조
- `App.js`: 메인 컴포넌트, 전체 상태 관리
- `TodoForm.js`: 할 일 입력 폼
- `TodoList.js`: 할 일 목록 렌더링
- `TodoItem.js`: 개별 할 일 아이템

## 실행 방법
